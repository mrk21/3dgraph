はじめに

OpenGLを用いた３次元グラフプロットソフトです。

このソフトは、テキストデータを読み込み、それを３次元グラフで表示するものです。 この手のソフトはgnuplotをはじめ, 多々存在しますが、 このソフトはグラフを斜投影でき、 必要に応じて、x軸ポリライン, y軸ポリラインをそれぞれ非表示にすることができます。

現在のところ、フルカラーでしか動作確認をしてませんし、 また、隠線処理が不完全ですが、今後対応する予定です。

インストール／アンインストール

このソフトはレジストリを使用していないため、コピーするだけで使用することができます。 不要になった時も普通に消去するだけで大丈夫です。

使用方法

3dGraph.exeを起動し、"ファイル->データを開く"でデータが記述されたテキストファイルを読み込むだけです。 ただし、テキストファイルは以下の条件を満たしていなければなりません。

  1. 各データが数値であること
  2. 各データが空白区切りであること
  3. データ配置が行列形式であること(列方向x軸、行方向y軸)

また、不正なデータを読み込んだときの動作は不定です。

option.iniについて

本ソフトは、描画オプションがありますが、 現在GUIでの設定項目はありません。 option.iniファイルを直接設定する必要があります（設定方法は、option.iniに記述してあります）。

現在有効な項目
  • 各座標軸の主・副目盛、各平面の主・副グリッド、x軸、y軸ポリラインの可視、非可視の設定
  • 各座標軸の主・副目盛の分割数（副目盛の分割数は主目盛に対してです）
  • 対数目盛にするかどうか
  • x軸、y軸方向の正規化(指定した座標軸方向の各ポリラインの面積を1にします)
  • 座標軸の向き(x軸、y軸のどちらが手前か)
